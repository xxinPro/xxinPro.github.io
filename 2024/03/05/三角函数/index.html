三角函数、反三角函数的基本属性与基本关系

三角函数

基本关系

正弦：$\sin A = \frac{a}{c} $  余弦：$\cos A = \frac{b}{c} $
正切：$\tan A = \frac{a}{b} $  余切：$\cot A = \frac{b}{a} $
余割：$\csc A = \frac{c}{a} $  正割：$\sec A = \frac{c}{b} $

恒等式

  1. $ \sec A = \frac{1}{\cos A} $
  2. $ \csc A = \frac{1}{\sin A} $
  3. $ \cot A = \frac{1}{\tan A} $
  4. $ \sin ^{2}A + \cos ^{2}A = 1 $
  5. $ \tan ^{2}A + 1 = \sec ^{2}A $
  6. $ \tan A = \frac{\sin A}{\cos A} $
  7. $ \cos 2A = \cos ^{2} A - \sin ^{2} A = 1 - 2\sin ^{2} A = 2\cos ^{2} A - 1 $
  8. $ \sin 2A = 2\sin A \times \cos A $
  9. $ \csc ^{2} A = \frac{1}{\sin ^{2} A} = \frac{\cos ^{2} A + \sin ^{2} A}{\sin ^{2} A} = \frac{\cos ^{2} A}{\sin ^{2} A} + 1 = \cot ^{2}A + 1 $
  10. $ \cos (\alpha \pm \beta ) = \cos \alpha \cos \beta \mp \sin \alpha \sin \beta $
  11. $ \sin (\alpha \pm \beta ) = \sin \alpha \cos \beta \pm \cos \alpha \sin \beta $
