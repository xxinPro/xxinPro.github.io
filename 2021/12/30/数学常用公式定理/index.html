三角函数公式

正弦：$\sin A = \frac{a}{c} $    余弦：$\cos A = \frac{b}{c} $
正切：$\tan A = \frac{a}{b} $   余切：$\cot A = \frac{b}{a} $
正割：$\sec A = \frac{c}{b} $   余割：$\csc A = \frac{c}{a} $

  

基本关系

  1. $ \sec A = \frac{1}{\cos A} $
  2. $ \csc A = \frac{1}{\sin A} $
  3. $ \cot A = \frac{1}{\tan A} $
  4. $ \sin ^{2}A + \cos ^{2}A = 1 $
  5. $ \tan ^{2}A + 1 = \sec ^{2}A $
  6. $ \tan A = \frac{\sin A}{\cos A} $
  7. $ \cos 2A = \cos ^{2} A - \sin ^{2} A = 1 - 2\sin ^{2} A = 2\cos ^{2} A - 1 $
  8. $ \sin 2A = 2\sin A \times \cos A $
  9. $ \csc ^{2} A = \frac{1}{\sin ^{2} A} = \frac{\cos ^{2} A + \sin ^{2} A}{\sin ^{2} A} = \frac{\cos ^{2} A}{\sin ^{2} A} + 1 = \cot ^{2}A + 1 $
  10. $ \cos (\alpha \pm \beta ) = \cos \alpha \cos \beta \mp \sin \alpha \sin \beta $
  11. $ \sin (\alpha \pm \beta ) = \sin \alpha \cos \beta \pm \cos \alpha \sin \beta $

和差化积公式

  1. $\sin \alpha + \sin \beta = 2\sin \frac{\alpha + \beta }{2} \cos \frac{\alpha - \beta }{2} $
  2. $\sin \alpha - \sin \beta = 2\cos \frac{\alpha + \beta }{2} \sin \frac{\alpha - \beta }{2} $
  3. $\cos \alpha + \cos \beta = 2\cos \frac{\alpha + \beta }{2} \cos \frac{\alpha - \beta }{2} $
  4. $\cos \alpha - \cos \beta = -2\sin \frac{\alpha + \beta }{2} \sin \frac{\alpha - \beta }{2} $
  5. $\tan \alpha + \tan \beta = \frac{\sin \left ( \alpha + \beta \right )}{\cos \alpha \cos \beta } $
  6. $\tan \alpha - \tan \beta = \frac{\sin \left ( \alpha - \beta \right )}{\cos \alpha \cos \beta } $
  7. $\cot \alpha + \cot \beta = \frac{\sin \left ( \alpha + \beta \right )}{\sin \alpha \sin \beta } $
  8. $\cot \alpha - \cot \beta = -\frac{\sin \left ( \alpha - \beta \right )}{\sin \alpha \sin \beta } $
  9. $\tan \alpha + \cot \beta = \frac{\cos \left ( \alpha - \beta \right )}{\cos \alpha \sin \beta } $
  10. $\tan \alpha - \cot \beta = -\frac{\cos \left ( \alpha + \beta \right )}{\cos \alpha \sin \beta } $

积化和差公式

  1. $\sin \alpha \cos \beta = \frac{1}{2} \left [ \sin (\alpha + \beta ) + \sin (\alpha - \beta ) \right]$

导数和微分公式

导数表达式结构

$
f’(x_{0} ) = \lim_{\Delta x \to 0} \frac{f(x_{0} + \Delta x) - f(x_{0} )}{\Delta x}
$

$
f’(x_{0} ) = \lim_{h \to 0} \frac{f(x_{0} + h) - f(x_{0} )}{h} (h = \Delta x)
$

$
f’(x_{0} ) = \lim_{x \to x_{0}} \frac{f(x) - f(x_{0} )}{x-x_{0}} (x = x_{0} + \Delta x)
$

导数基本公式

  1. $ C’ = 0(C为常数) $
  2. $ (x ^{n})’ = nx^{n-1}(n为实数) $
  3. $ (\log_{a}{x})’ = \frac{1}{x\ln_{}{a}} (a>0,a≠1) $
  4. $ (\ln_{}{x})’ = \frac{1}{x} $
  5. $ (a ^{x})’ = a^{x}\ln_{}{a}(a>0,a≠1) $
  6. $ (e^{x})’ = e^{x} $
  7. $ (\sin {x})’ = \cos {x} $
  8. $ (\cos {x})’ = -\sin {x} $
  9. $ (\tan {x})’ = \frac {1}{\cos ^2 {x}} = \sec ^2 {x} $
  10. $ (\cot {x})’ = -\frac{1}{\sin ^2 {x}} = -\csc ^2 {x} $
  11. $ (\sec {x})’ = \sec {x} \tan {x} $
  12. $ (\csc {x})’ = -\csc {x} \cot {x} $
  13. $ (\arcsin {x})’ = \frac {1}{\sqrt{1- x^2}} $
  14. $ (\arccos {x})’ = -\frac {1}{\sqrt{1- x^2}} $
  15. $ (\arctan {x})’ = \frac {1}{1+x^2} $
  16. $ (\operatorname{arccot} x)’ = -\frac {1}{1+x^2} $

切线与法线方程

函数$f(x)$在点$(x_{0},f(x_{0}))$处的切线与法线方程 $ \left\{\begin{matrix} (1).若f'(x_{0})≠0且f'(x_0)≠\infty ，\\ 切线方程为y-f(x_0) = f'(x_0)(x-x_0)，\\ 法线方程为y-f(x_0) = -\frac{1}{f'(x_0)}(x-x_0);\\ (2).若f'(x_{0})=0,切线方程为y=f(x_0),\\ 法线方程为x=x_0;\\ (3).若f'(x_{0})=\infty,切线方程为x=x_0,\\法线方程为y=f(x_0) \end{matrix}\right. $

定积分

定积分的性质

被积函数中的常数因子可以提到积分号外

$ $ \begin{align} \int_{a}^{b} kf(x)\mathrm{d}x & = k\int_{a}^{b} f(x)\mathrm{d}x \; (k是常数) \end{align} $ $

两个函数代数和、差的积分等于积分代数的和、差

$ $ \begin{align} \int_{a}^{b} [f(x) \pm g(x)]\mathrm{d}x = \int_{a}^{b} f(x)\mathrm{d}x \pm \int_{a}^{b} g(x) \mathrm{d}x \end{align} $ $

可加性，对于任意大小关系的a，b，c，有

$ $ \begin{align} \int_{a}^{b} f(x)\mathrm{d}x = \int_{a}^{c} f(x) \mathrm{d}x + \int_{c}^{b} f(x)\mathrm{d} x \end{align} $ $

如果在区间[a,b]上$f(x) \equiv 1$，则
$ $
\begin{align}
\int_{a}^{b} 1\mathrm{d}x = \int_{a}^{b} \mathrm{d}x = b-a
\end{align}
$ $

如果在区间[a,b]上$f(x) \ge 0$，则
$ $
\begin{align}
\int_{a}^{b} f(x)\mathrm{d}x \ge 0 \; (a<b)
\end{align}
$ $

如果在区间[a,b]上$f(x) \le g(x)$，则

$ $ \begin{align} \int_{a}^{b} f(x)\mathrm{d} x \le \int_{a}^{b} g(x)\mathrm{d} x \; (a < b) \end{align} $ $

$ $ \begin{align} |\int_{a}^{b}f(x)\mathrm{d}x | \le \int_{a}^{b} |f(x)| \mathrm{d} x \; (a < b) \end{align} $ $

估值定理，设M及m分别是函数$f(x)$在区间[a,b]上的最大值及最小值，则

$ $ \begin{align} m(b-a) \le \int_{a}^{b} f(x)\mathrm{d} x \le M(b-a) \; (a < b) \end{align} $ $

定积分中值定理，如果函数$f(x)$在积分区间[a,b]上连续，则在[a,b]上至少存在一个点$\xi $使下式成立，这个公式叫做积分中值公式

$ $ \begin{align} \int_{a}^{b} f(x)\mathrm{d}x = f(\xi )(b-a) \; (a \le \xi \le b) \end{align} $ $

由定积分的几何意义可知，当a>0时

$ $ \begin{align} \int_{-a}^{a} \sqrt{a^{2}-x^{2}} \mathrm{d}x = \frac{\pi }{2} a^{2} \\ \int_{0}^{a} \sqrt{a^{2}-x^{2}} \mathrm{d}x = \frac{\pi }{4} a^{2} \end{align} $ $

广义积分

广义积分的性质


如果$ \int_{a}^{+\infty } f(x)\mathrm{d}x $收敛，则$\int_{a}^{+\infty } kf(x)\mathrm{d}x $收敛，并且$\int_{a}^{+\infty } kf(x)\mathrm{d}x = k\int_{a}^{+\infty } f(x)\mathrm{d}x $


如果$ \int_{a}^{+\infty } f(x)\mathrm{d}x $与$ \int_{a}^{+\infty } g(x)\mathrm{d}x $收敛，则$ \int_{a}^{+\infty } [f(x) \pm g(x) ]\mathrm{d}x $收敛，反之不一定成立

广义积分的定理


如果$ f(x)\ge 0 $，那么$ \int_{a}^{+\infty } f(x)\mathrm{d}x $收敛的充分必要条件是，$ \int_{a}^{+\infty } f(x)\mathrm{d}x $的原函数$ F(x) = \int_{a}^{x} f(t)\mathrm{d}t $有界

设$ 0 \le f(x) \le g(x) $
如果$ \int_{a}^{+\infty } g(x)\mathrm{d}x $收敛，那么$ \int_{a}^{+\infty } f(x)\mathrm{d}x $收敛
如果$ \int_{a}^{+\infty } f(x)\mathrm{d}x $发散，那么$ \int_{a}^{+\infty } g(x)\mathrm{d}x $发散

定积分求导公式

设有$F(x)=\int_{g(x)}^{h(x)}f(t)\mathrm{d}t$

则它的导数$F’(x)$为

$ $ \begin{align} F'(x) & = h'(x)f[h(x)]-g'(x)f[g(x)] \end{align} $ $

二重积分

直角坐标下二重积分的计算

当D为X -型区域时

$ $ \begin{align} \iint\limits_{D}^{} f(x,y)\mathrm{d}x\mathrm{d}y = \int_{a}^{b}\mathrm{d}x\int_{\varphi_{1}(x)}^{\varphi_{2}(x)}f(x,y)\mathrm{d}y \end{align} $ $

当D为Y -型区域时

$ $ \begin{align} \iint\limits_{D}^{} f(x,y)\mathrm{d}x\mathrm{d}y = \int_{c}^{d}\mathrm{d}y\int_{\varphi_{1}(y)}^{\varphi_{2}(y)}f(x,y)\mathrm{d}x \end{align} $ $

极坐标系下二重积分的计算

直角坐标与极坐标的对应关系

因为
    $x=r\cos\theta,y=r\sin\theta$
于是
    $f(x,y)=f(r\cos\theta,r\sin\theta)$
因此
    $\iint\limits_{D}^{} f(x,y)\mathrm{d}x\mathrm{d}y = \iint\limits_{D}^{} f(r\cos\theta,r\sin\theta)r\mathrm{d}r\mathrm{d}\theta $

计算公式

极点在区域D内部时，$D=\{(\theta ,r)\, |\, 0\le \theta \le 2\pi ,0 \le r \le r(\theta )\}$，则有

$ $ \begin{align} \iint\limits_{D}^{} f(r\cos\theta,r\sin\theta)r\mathrm{d}r\mathrm{d}\theta = \int_{0}^{2\pi}\mathrm{d}\theta\int_{0}^{r(\theta)}f(r\cos\theta,r\sin\theta)r\mathrm{d}r \end{align} $ $

极点在区域D的边界曲线上，$D=\{(\theta ,r)\, |\, \alpha \le \theta \le \beta ,0 \le r \le r(\theta )\}$，则有

$ $ \begin{align} \iint\limits_{D}^{} f(r\cos\theta,r\sin\theta)r\mathrm{d}r\mathrm{d}\theta = \int_{\alpha }^{\beta}\mathrm{d}\theta\int_{0}^{r(\theta)}f(r\cos\theta,r\sin\theta)r\mathrm{d}r \end{align} $ $

极点在区域D之外，$D=\{(\theta ,r)\, |\, \alpha \le \theta \le \beta ,r_{1}(\theta)\le r \le r_2(\theta )\}$，则有

$ $ \begin{align} \iint\limits_{D}^{} f(r\cos\theta,r\sin\theta)r\mathrm{d}r\mathrm{d}\theta = \int_{\alpha }^{\beta}\mathrm{d}\theta\int_{r_{1}(\theta)}^{r_{2}(\theta)}f(r\cos\theta,r\sin\theta)r\mathrm{d}r \end{align} $ $

无穷级数

三大参照级数

几何级数（等比级数）

    其中$a \ne 0$，$q$是级数的公比

$ $ \begin{align} \sum_{n=1}^{\infty} aq^{n-1}=a+aq+aq^2+···+aq^{n-1}+··· \end{align} $ $

    当$|q|<1$时，级数收敛；当$|q|\ge1$时，级数发散

调和级数

    调和级数$\sum_{n=1}^{\infty } \frac{1}{n} $是发散的

$p$级数

$ $
\begin{align}
\sum_{n=1}^{\infty } \frac{1}{n^p}
\end{align}
$ $

    $p$级数当$0<p\le1$时是发散的，当$p>1$时是收敛的

数项级数

    如果级数的部分和数列$ \{S_n\} $有极限S（级数的和），则称级数是收敛的，如果没有极限，则称级数是发散的

收敛级数的性质


    若级数$\sum_{n=1}^{\infty } u_n$收敛于$S$，则级数$\sum_{n=1}^{\infty } ku_n$也收敛，且收敛于$kS$


    若级数$\sum_{n=1}^{\infty } u_n$与级数$\sum_{n=1}^{\infty } v_n$分别收敛于$S_1,S_2$，则级数$\sum_{n=1}^{\infty } (u_n \pm v_n)$收敛于$S_1 \pm S_2$；但级数$\sum_{n=1}^{\infty } u_n$与级数$\sum_{n=1}^{\infty } v_n$都发散时，级数$\sum_{n=1}^{\infty } (u_n \pm v_n)$不一定发散


    在级数中去掉、加上或改变有限项，不会改变该级数的敛散性


    如果级数收敛，则该级数的任意项加括号所成的级数仍收敛，且其和不变；如果加括号后所成的级数收敛，则不能断定原级数也收敛；如果加括号后的级数发散，则原级数也发散

级数收敛的必要条件


若级数$\sum_{n=1}^{\infty } u_n$收敛，则$\lim_{n \to \infty } u_n=0$


若$\lim_{n \to \infty } u_n \ne 0$或不存在，则级数$\sum_{n=1}^{\infty } u_n$发散

正项级数

正项级数的部分和数列$\{S_n\}$是单调增加的，所以
正项级数$\sum_{n=1}^{\infty } u_n$收敛的充分必要条件是它的部分和数列$\{S_n\}$有界


设$\sum_{n=1}^{\infty } u_n$和$\sum_{n=1}^{\infty } v_n$均为正项级数，且$u_n \le v_n$，

  1. 如果级数$\sum_{n=1}^{\infty } v_n$收敛，则级数$\sum_{n=1}^{\infty } u_n$收敛
  2. 如果级数$\sum_{n=1}^{\infty } u_n$发散，则级数$\sum_{n=1}^{\infty } v_n$发散


设$\sum_{n=1}^{\infty } u_n$和$\sum_{n=1}^{\infty } v_n$均为正项级数，且$\lim_{n \to \infty } \frac{u_n}{v_n} = A$

  1. 若$0<A<+\infty$，则$\sum_{n=1}^{\infty } u_n$和$\sum_{n=1}^{\infty } v_n$同时收敛或发散
  2. 若$A=0$，且$\sum_{n=1}^{\infty } v_n$收敛，则$\sum_{n=1}^{\infty } u_n$收敛
  3. 若$A=+\infty$且$\sum_{n=1}^{\infty } v_n$发散，则$\sum_{n=1}^{\infty } u_n$发散


设$\sum_{n=1}^{\infty } u_n$为正项级数，如果$\lim_{n \to \infty } \frac{u_n+1}{u_n} = \rho $

  1. 当$\rho < 1$时，级数收敛
  2. 当$\rho > 1$时，级数发散
  3. 当$\rho = 1$时，级数可能收敛，也可能发散


设$\sum_{n=1}^{\infty } u_n$为正项级数，如果$\lim_{n \to \infty } \sqrt[n]{u_n}=p$

  1. 当$p < 1$时，级数收敛
  2. 当$p > 1$时，级数发散
  3. 当$p=1$时，可能收敛，也可能发散


设$\sum_{n=1}^{\infty } u_n$为正项级数

  1. 如果$\lim_{n \to \infty} nu_n=l>0$，则级数发散
  2. 如果$p > 1$，而$\lim_{n \to \infty} n^pu_n=l(0\le l<+\infty)$，则级数收敛
交错级数


如果交错级数$\sum_{n=1}^{\infty } (-1)^{n-1}u_n$满足条件

  1. $u_n \ge u_{n+1}$
  2. $\lim_{n \to \infty} u_n=0$
    则级数收敛，且其和$S \le u_1$，其余项$r_n$的绝对值$|r_n| \le u_{n+1}$
任意项级数


如果级数$\sum_{n=1}^{\infty } |u_n|$收敛，则级数$\sum_{n=1}^{\infty } u_n$也收敛，此时称级数$\sum_{n=1}^{\infty } u_n$绝对收敛
如果级数$\sum_{n=1}^{\infty } |u_n|$发散，而级数$\sum_{n=1}^{\infty } u_n$也收敛，此时称级数$\sum_{n=1}^{\infty } u_n$条件收敛


如果级数$\sum_{n=1}^{\infty } |u_n|$收敛，则级数$\sum_{n=1}^{\infty } u_n$一定收敛

  1. 如果级数$\sum_{n=1}^{\infty } u_n$发散，则级数$\sum_{n=1}^{\infty } |u_n|$一定发散
  2. 当$\sum_{n=1}^{\infty } u_n$条件收敛时，级数$\sum_{n=1}^{\infty} \frac{|u_n|+u_n}{2}$和$\sum_{n=1}^{\infty} \frac{|u_n|-u_n}{2} $均发散
