一、基本概念

(一)、绝对值

$ $
\begin{align}
\left | a \right | & = \begin{Bmatrix} \notag
a & a\ge 0\
-a & a< 0
\end{Bmatrix}\quad \quad (\left | a \right | \ge 0)
\end{align}
$ $

绝对值不等式

$ $
\begin{align}
& \left | x \right | \le a \quad (a>0) \Longleftrightarrow -a \le x \le a; \notag \
& \left | x \right | \ge a \quad (a>0) \Longleftrightarrow x \ge a 或 x \le -a; \notag
\end{align}
$ $

(二)、区间

是介于某两个实数之间的全体实数，这两个实数叫做区间的端点

$ \left \{ x|a < x < b \right \} $称为$\color{red}{\text{开区间}}$，记作$ (a,b) $


$\left \{ x|a \le x \le b \right \} $称为$\color{red}{\text{闭区间}}$，记作$ [a,b] $

$ \left \{ x|a \le x < b \right \} $称为半开半闭区间，记作$ [a,b) $
$ \left \{ x|a < x \le b \right \} $称为半开半闭区间，记作$ (a,b] $

$ [a,+\infty ) = \left \{ x \mid a \le x \right \} $
$ \left ( -\infty ,b \right ) =\left \{ x \mid x < b \right \} $



  区间长度的定义:（上限-下限）
  两端点间的距离(线段的长度)称为区间的长度

练习

用区间表示下列不等式的解

$ $
\begin{align}
& x^2 < 9\
解: \notag \
& \left ( -\infty ,-3 \right ) \cup \left ( 3 ,+\infty \right ) \notag \
\notag \
& \left | x-2<1 \right | \
解: \notag \
& \left ( 1,3 \right ) \notag \
\notag \
& x^2-5x-6<0 \
解: \notag \
& \left ( -1,6 \right ) \notag
\end{align}
$ $

邻域




  设a与δ是两个实数，且δ>0
  数集$\left { x\mid \left | x-a \right | < \delta \right }$称为点a的δ邻域，点a叫做邻域的中心，δ叫做邻域的半径
$ $
\begin{align}
U_{\delta }{(a)} = \left { x\mid a-\delta <x<a+\delta \right }\notag
\end{align}
$ $

  点a的去心邻域，记作$U{\delta }^{0}{(a)}$
$ $
\begin{align}
U{\delta }^{0}{(a)} = \left { x\mid 0 < \left | x-a \right | < \delta \right }\notag
\end{align}
$ $
