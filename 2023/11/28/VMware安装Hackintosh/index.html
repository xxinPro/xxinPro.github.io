VMware安装Hackintosh

发表于 2023-11-28 分类于 菜鸡手记 本文字数： 1.1k
虚拟机黑苹果安装过程

黑苹果版本：macOS Monterey 12.5 (21G72)
VMware版本：VMware Workstation 16 Pro (16.2.4)

添加macos引导

  VMware是没有macos引导的，没有macos引导就无法创建macos系统的虚拟机
  以管理员身份运行Hackintosh中的Unlocker.exe，为VMware添加macos引导

  一般情况下它会自动读取VMware的路径，并自动选择工具下载位置，也可以手动选择，确定路径后点击Patch

  进度条结束之后，选定工具下载的路径下多了两个文件夹，VMware新建虚拟机也已经支持macos

添加虚拟机

  在VMware中创建新虚拟机，选择典型→稍后选择操作系统→macOS 12，然后下一步

  选择虚拟机位置没啥好说的，然后指定磁盘容量，并选择将虚拟磁盘储存为单个文件，然后下一步

  选择自定义硬件→新 CD/DVD (SATA)→使用ISO镜像文件，选择黑苹果镜像文件的位置，之后点击完成就好了

启动虚拟机

  启动macos虚拟机

  这说明安装成功了，然后就是选择语言

  选中磁盘工具，点击继续

  选中虚拟机分配的磁盘并抹掉

  名称可以随便改，其它的不要动，然后点击抹掉，结束之后左上角会出现“🔴”，点击它关闭这个页面

  选中安装macOS Monterey，并点击继续

  之后选择磁盘，继续

  这个过程很长，等待安装结束后会出现选择国家地区，辅助功能，输入法等，都是无所谓的东西，能跳过就跳过

  创建一个本地账户，然后是定位服务，虚拟机要个🐦定位服务，然后选择时区，都是不重要的

  成功进入系统

安装VMwareTools

  VMwareTools安装后可以和电脑之间传送文件，共享粘贴板等
  鼠标右击桌面的Install macOS Monterey，点击推出Install macOS Monterey

  点击菜单栏中的虚拟机→安装VMware Tools

  接着双击安装，就可以开始安装了

  输入刚刚创建账户的密码，就可以开始安装了

  安装完成之后重启一下虚拟机，就可以使用了
