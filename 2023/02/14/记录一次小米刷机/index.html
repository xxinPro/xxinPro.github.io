记录一次小米刷机

发表于 2023-02-14 分类于 菜鸡手记 本文字数： 1k
红米 NOTE 4X线刷

目标机型：Redmi Note 4X

准备工作

  1. 申请解锁bl并下载解锁工具：http://www.miui.com/unlock/index.html
  2. MiFlash下载：https://cdn.alsgp0.fds.api.mi-img.com/micomm/MiFlash2020-3-14-0.rar
  3. ROM包下载：https://roms.miuier.com/zh-cn/devices/mido/

解锁BL

  1.进入手机设置 → 开发者选项 → 设备解锁状态，如果显示“当前设备已锁定，手机数据处于安全状态”，说明设备没有解锁

  2.先将手机号与设备绑定，关闭wifi打开数据网络，点击底部“绑定账号和设备”

  3.下载BL解锁工具：http://www.miui.com/unlock/index.html

  4.解压缩并启动miflash_unlock.exe，并使用刚刚绑定的手机号登陆

  5.手机关机后，长按“开机键”和“音量减”，进入fastboot页面，并使用数据线连接电脑

  6.在解锁工具中点击右上角设置，点击驱动检测，给手机安装驱动

  7.驱动安装完毕后，点击解锁 → 确认解锁，注意：整个过程中务必确保数据线的连接

  8.解锁成功后，点击重启手机

  9.开机后进入手机设置 → 开发者选项 → 设备解锁状态，此时可以看到，设备已经解锁成功

开始刷机

  1.下载MiFlash，下载目标ROM，在MiFlash中选中解压后的ROM包

  2.手机进入fastboot页面，数据线连接电脑，然后点击“加载设备”，确定设备成功加载后点击“刷机”

  3.刷机过程中，千万不要断开数据线连接

  4.成功后重启手机即可
