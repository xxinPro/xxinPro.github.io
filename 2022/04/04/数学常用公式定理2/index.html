不定积分

不定积分的性质

设函数$f(x)$及$g(x)$的原函数存在，则

$ $ \begin{align} \int \left [ f(x) \pm g(x) \right ] \mathrm{d}x = \int f(x) \mathrm{d}x \pm \int g(x) \mathrm{d}x \end{align} $ $

设函数$f(x)$的原函数存在，$k$为非零常数，则
$ $
\begin{align}
\int kf(x)dx = k\int f(x)dx
\end{align}
$ $


$ $
\begin{align}
{\left ( \int f(x)dx \right ) }’ = f(x)
\end{align}

\begin{align}
d\int f(x)dx = f(x)dx
\end{align}
$ $


$ $
\begin{align}
\int {f}’(x)dx = f(x)+C
\end{align}

\begin{align}
\int df(x) = f(x)+C
\end{align}
$ $

三角代换

一般的，a>0时，当被积函数有

  1. $\sqrt{a^{2} - x^{2} } $，可做代换$x=a\sin {u}(-\frac{\pi }{2} < u < \frac{\pi }{2} )$
  2. $\sqrt{x^{2} + a^{2} } $，可做代换$x=a\tan {u}(-\frac{\pi }{2} < u < \frac{\pi }{2} )$
  3. $\sqrt{x^{2} - a^{2} } $，可做代换$ x=a\sec {u}(x>a时，0<u<\frac{\pi}{2}；x<-a时，令x=t，则t \gt a) $

基本积分公式

  1. $\int k\mathrm{d}x = kx+C $
  2. $\int x^{n}\mathrm{d}x=\frac{x^{n +1}}{n +1}+C, \left({n \neq -1}\right) $
  3. $\int \frac{1}{x}\mathrm{d}x= \ln \left| x \right| +C $
  4. $\int \frac{1}{x^{2} }\mathrm{d}x= -\frac{1}{x} +C $
  5. $\int e^{x}\mathrm{d}x= e^{x} + C $
  6. $\int a^{x}\mathrm{d}x=\frac{a^{x}}{\ln_{}{a} }+C \,\,\, \left({a > 0\text{且}a≠1 }\right) $
  7. $\int \cos x \mathrm{d}x = \sin x + C$
  8. $\int \sin x \mathrm{d}x = -\cos x + C $
  9. $\int \tan {x} \mathrm{d}x = -\ln |\cos {x}| + C $
  10. $\int \cot {x}\mathrm{d}x = \ln |\sin {x}| + C $
  11. $\int \sec {x}\mathrm{d}x = \ln |\sec {x} + \tan {x}| + C $
  12. $\int \csc {x}\mathrm{d}x = \ln |\cot {x} - \csc {x}| + C $
  13. $\int \frac{\mathrm{d}x }{\sqrt{1-x^{2}} } = \arcsin x + C$
  14. $\int -\frac{\mathrm{d}x }{\sqrt{1-x^{2}} } = \arccos x + C$
  15. $\int \frac{\mathrm{d}x }{1+x^{2} } = \arctan x + C$
  16. $\int -\frac{\mathrm{d}x }{1+x^{2} } = \operatorname{arccot} x + C$
  17. $\int \frac{\mathrm{d}x }{\cos ^{2}x} = \int \sec ^{2}x\mathrm{d}x = \tan x + C$
  18. $\int \frac{\mathrm{d}x }{\sin ^{2}x} = \int \csc ^{2}x\mathrm{d}x = -\cot x + C$
  19. $\int \sec x \tan x \mathrm{d}x = \sec x + C $
  20. $\int \csc x \cot x \mathrm{d}x = -\csc x + C $

常用凑微分等式

  1. $\mathrm{d}x = \frac{1}{a} \mathrm{d} (ax) $
  2. $x\mathrm{d}x = \frac{1}{2} \mathrm{d} (x^2) $
  3. $x^{2}\mathrm{d}x = \frac{1}{3} \mathrm{d} (x^3) $
  4. $\frac{\mathrm{d}x }{\sqrt{x} } = 2\mathrm{d} (\sqrt[]{x} )$
  5. $\frac{1}{x} \mathrm{d}x = \mathrm{d} (\ln \left | x \right | )$
  6. $\frac{1}{x^{2}} \mathrm{d}x = \mathrm{d} (-\frac{1}{x} )$
  7. $e^{x} \mathrm{d}x = \mathrm{d}(e^{x}) $
  8. $\cos x \mathrm{d}x = \mathrm{d}(\sin x) $
  9. $\sin x \mathrm{d}x = \mathrm{d}(-\cos x) $
  10. $\frac{1}{1+x^{2}} \mathrm{d}x = \mathrm{d}(\arctan x) $
  11. $\frac{1}{\sqrt[]{1-x^{2}} } \mathrm{d}x = \mathrm{d}(\arcsin x) $
  12. $\sec ^{2} x \mathrm{d}x = \mathrm{d}(\tan x) $
  13. $\csc ^{2} x \mathrm{d}x = \mathrm{d}(-\cot x) $

定积分

定积分的性质

当$ b=a $时

$ $ \begin{align} \int_{a}^{b} f(x)\mathrm{d}x = 0 \end{align} $ $

当$ a\gt b $时

$ $ \begin{align} \int_{a}^{b} f(x)\mathrm{d}x = -\int_{b}^{a} f(x)\mathrm{d}x \end{align} $ $

被积函数中的常数因子可以提到积分号外

$ $ \begin{align} \int_{a}^{b} kf(x)\mathrm{d}x = k\int_{a}^{b} f(x)\mathrm{d}x \; (k是常数) \end{align} $ $

两个函数代数和、差的积分等于积分代数的和、差

$ $ \begin{align} \int_{a}^{b} [f(x) \pm g(x)]\mathrm{d}x = \int_{a}^{b} f(x)\mathrm{d}x \pm \int_{a}^{b} g(x) \mathrm{d}x \end{align} $ $

可加性，设$ a \lt b \lt c $，有

$ $ \begin{align} \int_{a}^{b} f(x)\mathrm{d}x = \int_{a}^{c} f(x) \mathrm{d}x + \int_{c}^{b} f(x)\mathrm{d} x \end{align} $ $

如果在区间$[a,b]$上$f(x) \equiv 1$，则

$ $ \begin{align} \int_{a}^{b} 1\mathrm{d}x = \int_{a}^{b} \mathrm{d}x = b-a \end{align} $ $

如果在区间$[a,b]$上$f(x) \ge 0$，则

$ $ \begin{align} \int_{a}^{b} f(x)\mathrm{d}x \ge 0 \; (a \lt b) \end{align} $ $

如果在区间$[a,b]$上$f(x) \le g(x)$，则

$ $ \begin{align} \int_{a}^{b} f(x)\mathrm{d} x \le \int_{a}^{b} g(x)\mathrm{d} x \; (a \lt b) \end{align} $ $

$ $ \begin{align} |\int_{a}^{b}f(x)\mathrm{d}x | \le \int_{a}^{b} |f(x)| \mathrm{d} x \; (a \lt b) \end{align} $ $

估值定理。设$M$及$m$分别是函数$f(x)$在区间$[a,b]$上的最大值及最小值，则

$ $ \begin{align} m(b-a) \le \int_{a}^{b}f(x)\mathrm{d}x \le M(b-a) \end{align} $ $

定积分中值定理。如果函数$f(x)$在积分区间$[a,b]$上连续，那么在$[a,b]$上至少存在一个点$\xi$，使下式成立

$ $ \begin{align} \int_{a}^{b}f(x)\mathrm{d}x=f(\xi)(b-a) \; (a \le \xi \le b) \end{align} $ $

由定积分的几何意义可知，当$a\gt 0$时

$ $ \begin{align} \int_{-a}^{a} \sqrt{a^{2}-x^{2}} \mathrm{d}x = \frac{\pi }{2} a^{2} \\ \int_{0}^{a} \sqrt{a^{2}-x^{2}} \mathrm{d}x = \frac{\pi }{4} a^{2} \end{align} $ $

牛顿-莱布尼兹公式

如果函数$F(x)$是连续函数$f(x)$在区间$[a,b]$上的一个原函数，那么

$ $ \begin{align} \int_{a}^{b}f(x)\mathrm{d}x=F(b) -F(a) \end{align} $ $

广义积分

广义积分的性质


如果$ \int_{a}^{+\infty } f(x)\mathrm{d}x $收敛，则$\int_{a}^{+\infty } kf(x)\mathrm{d}x $收敛，并且$\int_{a}^{+\infty } kf(x)\mathrm{d}x = k\int_{a}^{+\infty } f(x)\mathrm{d}x $


如果$ \int_{a}^{+\infty } f(x)\mathrm{d}x $与$ \int_{a}^{+\infty } g(x)\mathrm{d}x $收敛，则$ \int_{a}^{+\infty } [f(x) \pm g(x) ]\mathrm{d}x $收敛，反之不一定成立

广义积分的定理


如果$ f(x)\ge 0 $，那么$ \int_{a}^{+\infty } f(x)\mathrm{d}x $收敛的充分必要条件是，$ \int_{a}^{+\infty } f(x)\mathrm{d}x $的原函数$ F(x) = \int_{a}^{x} f(t)\mathrm{d}t $有界

设$ 0 \le f(x) \le g(x) $
如果$ \int_{a}^{+\infty } g(x)\mathrm{d}x $收敛，那么$ \int_{a}^{+\infty } f(x)\mathrm{d}x $收敛
如果$ \int_{a}^{+\infty } f(x)\mathrm{d}x $发散，那么$ \int_{a}^{+\infty } g(x)\mathrm{d}x $发散

定积分求导公式

设有$F(x)=\int_{g(x)}^{h(x)}f(t)\mathrm{d}t$

则它的导数$F’(x)$为

$ $ \begin{align} F'(x) = h'(x)f[h(x)]-g'(x)f[g(x)] \end{align} $ $

微分方程

定义

  定义1：凡含有参数，未知函数和未知函数导数(或微分)的方程，称为微分方程，有时简称为方程，未知函数是一元函数的微分方程称作常微分方程，未知函数是多元函数的微分方程称作偏微分方程。微分方程中出现的未知函数最高阶导数的阶数，称为微分方程的阶。

  定义2：任何代入微分方程后使其成为恒等式的函数，都叫做该方程的解。若微分方程的解中含有任意常数的个数与方程的阶数相同，且任意常数之间不能合并，则称此解为该方程的通解(或一般解)。当通解中的各任意常数都取特定值时所得到的解，称为方程的特解。

一阶线性微分方程

非齐次

  一阶非齐次线性微分方程$ $
\begin{align}
\frac{\mathrm{d}y}{\mathrm{d}x}+P(x)y=Q(x)
\end{align}
$ $

  一阶非齐次线性微分方程的通解

$ $ \begin{align} y=e^{-\int P(x)\mathrm{d}x}[\int Q(x)e^{\int P(x)\mathrm{d}x}\mathrm{d}x+C] \end{align} $ $

齐次

  一阶齐次线性微分方程$ $
\begin{align}
\frac{\mathrm{d}y}{\mathrm{d}x}+P(x)y=0
\end{align}
$ $

  一阶齐次线性微分方程的通解$ $
\begin{align}
y=Ce^{-\int P(x)\mathrm{d}x} \; (C=\pm e^{C_1})
\end{align}
$ $

高阶线性微分方程

常系数齐次线性微分方程

  形如$ \eqref {eq36} $称为常系数齐次线性微分方程，其中$p$、$q$为常数

$ $
\begin{align} \label {eq36}
y’’+py’+qy = 0
\end{align}
$ $

常系数齐次线性微分方程的通解

  根据$ \eqref {eq36} $列出特征方程，如一元二次方程$ \eqref {eq37} $
$ $
\begin{align}\label {eq37}
r^2+pr+q=0
\end{align}
$ $

  解方程$ \eqref {eq37} $，有如下三种情况

1.当$\Delta = p^2-4q \gt 0$时，特征方程$ \eqref {eq37} $有两个不相等的实数根$r_1,r_2$$ $

\begin{align}
r_1 = \frac{-p+\sqrt{\Delta } }{2} \\
r_2 = \frac{-p-\sqrt{\Delta } }{2}
\end{align}
$ $

  此时微分方程$ \eqref {eq36} $的通解是$ $
\begin{align}
y=C_1e^{r_1x}+C_2e^{r_2x}
\end{align}
$ $

2.当$\Delta = p^2-4q=0$时，特征方程$ \eqref {eq37} $有两个相等的实数根$r_1=r_2=-\frac{p}{2} $

  此时微分方程$ \eqref {eq36} $的通解是$ $
\begin{align}
y=(C_1+C_2x)e^{r_1x}
\end{align}
$ $

3.当$\Delta = p^2-4q \lt 0$时，特征方程$ \eqref {eq37} $有一对共轭的复根$r_1,r_2$（$\alpha,\beta$均为实数,$\beta\ne0$），$i$为虚数，一般等于$\sqrt[]{-1} $

$ $
\begin{align}
r_1 = \frac{-b+\sqrt[]{\Delta } }{2} = \alpha +i\beta \\
r_2 = \frac{-b-\sqrt[]{\Delta } }{2} = \alpha - i\beta
\end{align}
$ $

  此时微分方程$ \eqref {eq36} $的通解是
$ $
\begin{align}
y=e^{\alpha x}(C_1\cos {\beta x}+C_2\sin {\beta x})
\end{align}
$ $

常系数非齐次线性微分方程

  形如$ \eqref {eq38} $称为常系数非齐次线性微分方程，其中$p$、$q$为常数
$ $
\begin{align} \label {eq38}
y’’+py’+qy = f(x)
\end{align}
$ $

常系数非齐次线性微分方程的通解

  形如如下两种形式的常系数非齐次线性微分方程可以使用待定系数法求特解

$ $
\begin{align}
f(x)=e^{\lambda x}P_n(x)
\end{align}
$ $

$ $ \begin{align} f(x)=e^{\lambda x}[P_l(x)\cos (\omega x)+Q_n(x)\sin (\omega x)] \end{align} $ $

  对应的原方程特解待定式为

$ $
\begin{align}
y_0(x) = x^ke^{\lambda x}R_m(x)
\end{align}
$ $

$ $
\begin{align}
y_0(x) = x^ke^{\lambda x}[R_{m1}(x)\cos \omega x+R_{m2}\sin \omega x]
\end{align}
$ $

二重积分

直角坐标下二重积分的计算

当D为X -型区域时

$ $ \begin{align} \iint\limits_{D}^{} f(x,y)\mathrm{d}x\mathrm{d}y = \int_{a}^{b}\mathrm{d}x\int_{\varphi_{1}(x)}^{\varphi_{2}(x)}f(x,y)\mathrm{d}y \end{align} $ $

当D为Y -型区域时

$ $ \begin{align} \iint\limits_{D}^{} f(x,y)\mathrm{d}x\mathrm{d}y = \int_{c}^{d}\mathrm{d}y\int_{\varphi_{1}(y)}^{\varphi_{2}(y)}f(x,y)\mathrm{d}x \end{align} $ $

极坐标系下二重积分的计算

直角坐标与极坐标的对应关系

因为
  $x=r\cos\theta,y=r\sin\theta$
于是
  $f(x,y)=f(r\cos\theta,r\sin\theta)$
其中
  $r=r(\theta )$
因此

$ $ \begin{align} \iint\limits_{D}^{} f(x,y)\mathrm{d}x\mathrm{d}y = \iint\limits_{D}^{} f(r\cos\theta,r\sin\theta)r\mathrm{d}r\mathrm{d}\theta \end{align} $ $

计算公式

极点在区域D内部时，$D=\{(\theta ,r)\, |\, 0 \le \theta \le 2\pi ,0 \le r \le r(\theta )\}$，则有 $ $ \begin{align} \iint\limits_{D}^{} f(r\cos\theta,r\sin\theta)r\mathrm{d}r\mathrm{d}\theta = \int_{0}^{2\pi}\mathrm{d}\theta\int_{0}^{r(\theta)}f(r\cos\theta,r\sin\theta)r\mathrm{d}r \end{align} $ $
极点在区域D的边界曲线上，$D=\{(\theta ,r)\, |\, \alpha \le \theta \le \beta ,0 \le r \le r(\theta )\}$，则有 $ $ \begin{align} \iint\limits_{D}^{} f(r\cos\theta,r\sin\theta)r\mathrm{d}r\mathrm{d}\theta = \int_{\alpha }^{\beta}\mathrm{d}\theta\int_{0}^{r(\theta)}f(r\cos\theta,r\sin\theta)r\mathrm{d}r \end{align} $ $
极点在区域D之外，$D=\{(\theta ,r)\, |\, \alpha \le \theta \le \beta ,r_{1}(\theta) \le r \le r_2(\theta )\}$，则有 $ $ \begin{align} \iint\limits_{D}^{} f(r\cos\theta,r\sin\theta)r\mathrm{d}r\mathrm{d}\theta = \int_{\alpha }^{\beta}\mathrm{d}\theta\int_{r_{1}(\theta)}^{r_{2}(\theta)}f(r\cos\theta,r\sin\theta)r\mathrm{d}r \end{align} $ $

无穷级数

三大参照级数

几何级数（等比级数）

    其中$a \ne 0$，$q$是级数的公比

$ $ \begin{align} \sum_{n=1}^{\infty} aq^{n-1}=a+aq+aq^2+···+aq^{n-1}+··· \end{align} $ $

    当$\left| q \right| \lt 1$时，级数收敛；当$\left|q \right|\ge1$时，级数发散

调和级数

    调和级数$\sum_{n=1}^{\infty } \frac{1}{n} $是发散的

$p$级数

$ $
\begin{align}
\sum_{n=1}^{\infty } \frac{1}{n^p}
\end{align}
$ $

    $p$级数当$0 \lt p \le 1$时是发散的，当$p\gt 1$时是收敛的

数项级数

    如果级数的部分和数列$ \{S_n\} $有极限S（级数的和），则称级数是收敛的，如果没有极限，则称级数是发散的

收敛级数的性质


    若级数$\sum_{n=1}^{\infty } u_n$收敛于$S$，则级数$\sum_{n=1}^{\infty } ku_n$也收敛，且收敛于$kS$


    若级数$\sum_{n=1}^{\infty } u_n$与级数$\sum_{n=1}^{\infty } v_n$分别收敛于$S_1,S_2$，则级数$\sum_{n=1}^{\infty } (u_n \pm v_n)$收敛于$S_1 \pm S_2$；但级数$\sum_{n=1}^{\infty } u_n$与级数$\sum_{n=1}^{\infty } v_n$都发散时，级数$\sum_{n=1}^{\infty } (u_n \pm v_n)$不一定发散


    在级数中去掉、加上或改变有限项，不会改变该级数的敛散性


    如果级数收敛，则该级数的任意项加括号所成的级数仍收敛，且其和不变；如果加括号后所成的级数收敛，则不能断定原级数也收敛；如果加括号后的级数发散，则原级数也发散

级数收敛的必要条件


若级数$\sum_{n=1}^{\infty } u_n$收敛，则$\lim_{n \to \infty } u_n=0$


若$\lim_{n \to \infty } u_n \ne 0$或不存在，
则级数$\sum_{n=1}^{\infty } u_n$发散

正项级数

正项级数的部分和数列$\{S_n\}$是单调增加的，所以
正项级数$\sum_{n=1}^{\infty } u_n$收敛的充分必要条件是它的部分和数列$\{S_n\}$有界


设$\sum_{n=1}^{\infty } u_n$和$\sum_{n=1}^{\infty } v_n$均为正项级数，且$u_n \le v_n$，

  1. 如果级数$\sum_{n=1}^{\infty } v_n$收敛，则级数$\sum_{n=1}^{\infty } u_n$收敛
  2. 如果级数$\sum_{n=1}^{\infty } u_n$发散，则级数$\sum_{n=1}^{\infty } v_n$发散


设$\sum_{n=1}^{\infty } u_n$和$\sum_{n=1}^{\infty } v_n$均为正项级数，且$\lim_{n \to \infty } \frac{u_n}{v_n} = A$

  1. 若$0 \lt A \lt +\infty$，则$\sum_{n=1}^{\infty } u_n$和$\sum_{n=1}^{\infty } v_n$同时收敛或发散
  2. 若$A=0$，且$\sum_{n=1}^{\infty } v_n$收敛，则$\sum_{n=1}^{\infty } u_n$收敛
  3. 若$A=+\infty$且$\sum_{n=1}^{\infty } v_n$发散，则$\sum_{n=1}^{\infty } u_n$发散


设$\sum_{n=1}^{\infty } u_n$为正项级数，如果$\lim_{n \to \infty } \frac{u_n+1}{u_n} = \rho $

  1. 当$\rho \lt 1$时，级数收敛
  2. 当$\rho \gt 1$时，级数发散
  3. 当$\rho = 1$时，级数可能收敛，也可能发散


设$\sum_{n=1}^{\infty } u_n$为正项级数，如果$\lim_{n \to \infty } \sqrt[n]{u_n}=p$

  1. 当$p \lt 1$时，级数收敛
  2. 当$p \gt 1$时，级数发散
  3. 当$p=1$时，可能收敛，也可能发散


设$\sum_{n=1}^{\infty } u_n$为正项级数

  1. 如果$\lim_{n \to \infty} nu_n=l \gt 0$，则级数发散
  2. 如果$p \gt 1$，而$\lim_{n \to \infty} n^pu_n=l(0\le l \lt +\infty)$，则级数收敛

交错级数


如果交错级数$\sum_{n=1}^{\infty } (-1)^{n-1}u_n$满足条件

  1. $u_n \ge u_{n+1}$
  2. $\lim_{n \to \infty} u_n=0$
    则级数收敛，且其和$S \le u_1$，其余项$r_n$的绝对值$\left|r_n\right| \le u_{n+1}$

任意项级数


  如果级数$\sum_{n=1}^{\infty } |u_n|$收敛，则级数$\sum_{n=1}^{\infty } u_n$也收敛，此时称级数$\sum_{n=1}^{\infty } u_n$绝对收敛
  如果级数$\sum_{n=1}^{\infty } |u_n|$发散，而级数$\sum_{n=1}^{\infty } u_n$也收敛，此时称级数$\sum_{n=1}^{\infty } u_n$条件收敛


如果级数$\sum_{n=1}^{\infty } |u_n|$收敛，则级数$\sum_{n=1}^{\infty } u_n$一定收敛

  1. 如果级数$\sum_{n=1}^{\infty } u_n$发散，则级数$\sum_{n=1}^{\infty } |u_n|$一定发散
  2. 当$\sum_{n=1}^{\infty } u_n$条件收敛时，级数$\sum_{n=1}^{\infty} \frac{|u_n|+u_n}{2}$和$\sum_{n=1}^{\infty} \frac{|u_n|-u_n}{2} $均发散

幂级数


    先求幂级数的收敛区间$(-R,R)$，然后讨论幂级数在$x=R$与$x=-R$处的敛散性，从而得出幂级数的收敛域


  如果幂级数$\sum_{n=0}^{\infty}a_nx^n$当$x=x_0(x_0\ne0)$时收敛，则对于所有满足$|x| \lt |x_0|$的点$x$，幂级数$\sum_{n=0}^{\infty}a_nx^n$绝对收敛，
  反之，如果幂级数$\sum_{n=0}^{\infty}a_nx^n$当$x=x_0(x_0\ne0)$时发散，则对于所有满足$|x| \gt |x_0|$的点$x$，幂级数$\sum_{n=0}^{\infty}a_nx^n$发散


设幂级数$\sum_{n=0}^{\infty}a_nx^n$，
有$\lim_{n \to \infty} |\frac{a_n+1}{a_n} | = \rho$，则

  1. 若$\rho \ne 0$，则收敛半径$R=\frac{1}{\rho}$
  2. 若$\rho = 0$，则收敛半径$R = +\infty$
  3. 若$\rho = +\infty$，则收敛半径$R=0$

幂级数四则运算

    设幂级数$\sum_{n=0}^{\infty}a_nx^n$在其收敛区间$(-R_1,R_1)(R_1 \gt 0)$内的和函数为$S_1(x)$，幂级数$\sum_{n=0}^{\infty}b_nx^n$在其收敛区间$(-R_2,R_2)(R_2 \gt 0)$内的和函数为$S_2(x)$，取$R=\mathrm{min}\{R_1,R_2\} $，则有

  1. 加减运算$\eqref {eq27}$

    $ $ \begin{align} \label {eq27} \sum_{n=0}^{\infty}a_nx^n \pm \sum_{n=0}^{\infty}b_nx^n=\sum_{n=0}^{\infty}(a_n \pm b_n)x^n = S_1(x) \pm S_2(x) \end{align} $ $
      其收敛区间为$(-R,R)$

  2. 乘法运算$\eqref {eq28}$

    $ $ \begin{align} \label {eq28} \sum_{n=0}^{\infty}a_nx^n \cdot \sum_{n=0}^{\infty}b_nx^n = a_0b_0 + (a_0b_1+a_1b_0)x+(a_0b_2+a_1b_1+a_2b_0)x^2+ \cdot \cdot \cdot + (a_0b_n+a_1b_{n-1}+a_2b_{n-2}+\cdot \cdot \cdot +a_nb_0)x^n+\cdot \cdot \cdot = S_1(x)S_2(x) \end{align} $ $
      其收敛区间为$(-R,R)$

幂函数的和函数的性质

    幂级数在其收敛区间内可以逐项求导或逐项积分，并且逐项求导或者逐项积分后所得幂级数的收敛区间不变，但在其收敛区间的端点处，级数的敛散性可能会改变

设幂级数$\sum_{n=0}^{\infty}a_nx^n$的和函数为$S(x)$，则

  1. $S(x)$在其收敛域上连续，当$x_0 \in (-R,R)$时，有

    $ $ \begin{align} \lim_{x \to x_0 } (\sum_{n=0}^{\infty } a_nx^n) = \sum _{n=0}^{\infty}(\lim_{x \to x_0 }a_nx^n) = \sum _{n=0}^{\infty}a_nx^n_0 = S(x_0) \end{align} $ $

  2. 微分运算，$S(x)$在其收敛区间$(-R,R)$内可导，且

    $ $ \begin{align} S'(x)=(\sum_{n=0}^{\infty}a_nx^n)'=\sum_{n=0}^{\infty}(a_nx^n)'=\sum_{n=0}^{\infty}na_nx^{n-1} \end{align} $ $
      其收敛区间为$(-R,R)$

  3. 积分运算，$S(x)$在其收敛域上可积，且

    $ $ \begin{align} \int_{0}^{x}S(t)\mathrm{d}t=\int_{0}^{x} (\sum_{n=0}^{\infty}a_nt^n)\mathrm{d}t=\sum_{n=0}^{\infty}(\int_{0}^{x}a_nt^n\mathrm{d}t)=\sum_{n=0}^{\infty} \frac{a_n}{n+1}x^{n+1} \end{align} $ $
      其收敛区间为$(-R,R)$
