eclipse搭建Android环境

发表于 2022-10-17 分类于 菜鸡手记 本文字数： 2.6k

下载eclipse

  刚一开始我下载的是最新版的eclipse，但是不支持java1.8，所以后面下载了最后一个支持java1.8的版本

  eclipse 2020-06版本

安装ADT

  这里我使用在线安装的方式（需要科学上网），在线安装地址：https://dl-ssl.google.com/android/eclipse

  点击安装新软件

  点击add

  然后在线安装，如果是本地安装则添加jar:file:/绝对路径!/

  下一步

  继续下一步

  完成

  接着右下角会有下载进度，这是一个极其缓慢的过程，期间需要保持科学上网

  过程中会弹出这个弹窗，点击仍要安装

SDK安装

  待ADT安装完毕后重启eclipse，然后安装SDK Tools、SDK Platform-Tools和一个android SDK

  点击安装

  安装完毕后，打开SDK Manager

  在SDK管理器中可以管理各个版本的SDK

  安装想要的SDK版本，system image是模拟器的镜像文件，占用内存空间较多，可以不下载

显示SDK Manager

  如果在window中不显示sdk manager选项，那么就需要手动添加它

手动安装SDK Tools

  我下载的android sdk tools好像都是阉割的，其android.bat根本无法启动sdk manager

  而后发现了一个安装版(点击下载)，安装之后的SDK Tools是完整版
